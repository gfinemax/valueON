# Value On (밸류온) 사용자 매뉴얼

비즈니스 수지 분석 시뮬레이터 'Value On'의 사용 방법을 안내합니다.

## 1. 시작하기
웹 브라우저에서 애플리케이션에 접속합니다. 좌측 **사이드바**를 통해 주요 페이지로 이동할 수 있습니다.

- **대시보드:** 수지 분석을 수행하는 메인 화면
- **원시 데이터:** 분석에 사용되는 초기 데이터(Excel/CSV) 확인
- **설정:** 앱의 인터페이스 및 편집 권한 설정

## 2. 대시보드 사용법

### 2.1 간편 모드 vs 상세 모드
대시보드 상단의 **모드 토글 스위치**를 사용하여 모드를 전환할 수 있습니다.
- **간편 모드:** 기본적인 평당가와 금리 설정만으로 빠르게 예측합니다.
- **상세 모드 (Advanced):** 토지비, 건축비, 금융비 등 각 항목을 개별적으로 상세하게 조정할 수 있습니다.

### 2.2 입력 및 수정
- **유닛 믹스 (Unit Mix):** 각 평형(Type)별 세대수와 타겟 분양가를 설정합니다.
- **비용 카테고리:**
  - 항목의 금액을 클릭하여 수정합니다.
  - **항목 추가:** 카드 하단의 '+ 항목 추가하기' 버튼을 누릅니다.
  - **카테고리 삭제:** 카테고리 제목 옆의 '휴지통' 아이콘을 클릭하여 전체 카테고리를 삭제할 수 있습니다.
  - **계산 기준 설정:** 각 항목의 좌측 배지를 클릭하여 계산 방식을 변경할 수 있습니다.
    - **사유지/국유지:** 프로젝트 정보의 사유지/국유지 면적을 기준으로 계산합니다.
    - **평형입력:** 특정 면적을 직접 입력하여 계산할 수 있습니다. (예: 1000평 * 단가)

### 2.3 결과 확인
화면 우측(모바일은 하단)의 **Summary Dashboard**에서 실시간 계산 결과를 확인하세요.
- 총 사업비, 평당 원가
- 평형별 예상 분담금 (59/84타입)
- 비용 구성 도넛 차트

## 3. 글로벌 검색 (Search)
헤더의 🔍 아이콘을 통해 모든 수지 항목을 빠르게 검색할 수 있습니다.

- **검색 범위:** 비용 카테고리, 세부 항목, 분양 타입, 분양가 항목
- **자동 이동:** 검색 결과 클릭 시 해당 페이지로 이동하며, 비용 항목의 경우 해당 카테고리가 자동으로 펼쳐지고 항목이 하이라이트됩니다.
- **키보드 조작:** 
  - `↑` / `↓`: 결과 탐색
  - `Enter`: 선택 항목으로 이동
  - `Esc`: 검색창 닫기

## 4. 설정 (Settings)
사이드바 하단의 **설정** 메뉴에서 앱의 동작 방식을 변경할 수 있습니다.

- **항목 이동 허용:** 활성화 시 지출 항목들의 순서를 드래그하여 자유롭게 변경할 수 있습니다.
- **카테고리 추가 허용:** 활성화 시 새로운 지출 카테고리나 항목을 추가할 수 있는 버튼이 나타납니다.
- **항목 삭제 허용:** 활성화 시 카테고리, 항목, 세부항목의 삭제(휴지통) 아이콘이 표시됩니다. 데이터 안정성을 위해 평소에는 꺼두는 것이 좋습니다.

## 5. 초기화
헤더 우측 상단의 **'초기화'** 텍스트를 클릭하면 모든 설정이 기본값으로 돌아갑니다.
